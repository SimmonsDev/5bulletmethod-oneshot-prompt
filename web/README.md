# 5BulletMethod Web Frontend

## Setup

1. Install dependencies:
   ```
   npm install
   ```
2. Start development server:
   ```
   npm run dev
   ```

## Features
- Weekly entry form (5 bullets)
- History view
- Streak tracker
- Motivational AI insight (stub)

## API
- Expects backend running at `/api` endpoints

## Styling
- Tailwind CSS
